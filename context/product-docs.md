# TechCorp Product Documentation

## Getting Started with TechCorp Core
TechCorp Core is where your project begins. Here's a quick guide:
1. **Create an account**: Sign up on our website with your email.
2. **Create a project**: Give it a name and description.
3. **Invite your team**: Add members via email.
4. **Create tasks**: Break down your project into manageable tasks.
5. **Set priorities**: Use labels to indicate task importance.

## Troubleshooting Common Issues
### Password Resets
If you've forgotten your password:
- Go to the login page and click "Forgot Password?".
- Enter your registered email.
- Check your inbox for a reset link.
- Follow the instructions in the email to set a new password.

### Integration Issues with GitHub
If your GitHub integration isn't working:
- Go to your project settings.
- Select the "Integrations" tab.
- Click "Disconnect" on the GitHub card.
- Click "Connect" again and authorize TechCorp.
- Ensure your GitHub account has the necessary permissions.

## TechCorp Insights Analytics
TechCorp Insights provides key metrics for your team:
- **Velocity**: The number of story points completed per sprint.
- **Burndown**: A chart showing remaining work vs. time.
- **Cycle Time**: The time it takes for a task to go from "To Do" to "Done".
